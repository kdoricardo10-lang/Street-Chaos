Hi it's Vairek, thanks for downloading my asset!

If you're just looking to drop it into your godot project
then simply drag and drop arming_sword.glb wherever you'd
like in your project directory.

A quick word on texture sizes, don't let the 4K resolution
deter you, this asset uses 1:4 ratio textures to save on size
and put the detail where it counts. So while I'm calling them 
4K/2K/1K, when compared to typical 1:1 ratio textures the 
performance cost is that of 2K/1K/512
